# Proyecto-Monitor-Cripto-Moneda
Sistema de Gestión para Guardería ADSw-AGUGU desarrollado en Java y JavaFX bajo el patrón MVC. Permite administrar niños, empleados, sucursales, pagos, actividades y vacunas, incorporando persistencia de datos mediante archivos TXT, CSV, JSON y serialización de objetos. Proyecto académico de Análisis y Diseño de Software.
