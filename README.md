# Proyecto-Monitor-Cripto-Moneda
Dispositivo IoT basado en ESP8266 que monitorea precios de criptomonedas en tiempo real usando MQTT y la API de Binance. Muestra el precio en pantalla OLED, indica variaciones con LEDs y emite alertas sonoras configurables remotamente desde una app Java. Proyecto universitario — Comunicaciones y Redes, PUJ 2026-10.
